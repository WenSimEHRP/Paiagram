# Project Transport

This application is for visualizing, analyzing, managing, and editing transport timetable diagrams.
The program uses Bevy and Egui for data management using ECS and GUI rendering, respectively.

## Folder Structure

- The `src` folder holds the main source code of the application.
- The `assets` folder contains static assets like images, fonts, and configuration files.
- `src/main.rs` is the entry point of the application.
