fn force_close(&mut self, _tab: &mut Self::Tab) -> bool

This is called every frame after ui is called, if the _tab is active.

Returns true if the tab should be forced to close, false otherwise.

In the event this function returns true the tab will be removed without calling on_close.

fn on_rect_changed(&mut self, _tab: &mut Self::Tab)

Called when the rectangle of the tab content changes.

This can happen when the window is resized, panels are docked or undocked, or when the layout of the dock area is changed in any way that affects the available space for the tab content.

This is useful for tabs that need to adjust their content based on the available space.
